格式化硬盘与自建常用创建与删命令

fdisk -l 命令查看刚刚接入USB的U盘/硬盘并记下其设备名称
查看挂载用这个下面
df -h

  1. mkfs.ext4 —— 创建ext4文件系统硬盘格式化。

mkfs.FAT32 /dev/sda1 （这是我的分区U盘名/dev/mmcblk1p1）
mkfs.ext4 /dev/mmcblk0p1

四、用blkid 命令查看所需要挂载硬盘的UUID，并复制保存UUID。例如下图：
blkid /dev/sda1
这是我的

F230-4E90” BLOCK_SIZE=”512” TYPE=”vfat” PARTUUID=
利用cd命令切换到要挂载的目标目录，然后利用mkdir命令创建挂载目录 ，也可以使用WinSCP软件直接目标目录下创建挂载目录。例如下图：
进入mnt目录下创建一个文件夹，权根0755

六、打开/etc/fstab文件，在最后行追加（记住是追加，原有的内容不要动）下面内容：
mnt下面建一个libaob2 新建的目录
UUID=0f54da88-7f6d-4f41-9069-8b9ce53c045f /mnt/libao2/ ext4 defaults 0 0

reboot重启后会看到mnt下面建一个了一个libaob2 新建的目录了

CD到指定目录

cd 命令切换目录只需在命令后加目录名称便可。例如：测试

ln-s /mnt/libao2/www/ /www 软连

cd /mnt/libao2/

下面去libao3文件夹
cd /mnt/libao2/libao3/
创建和删软链
#ln –s /mnt/libao2/www/ /wwww//

#rm –rf www

#rm –rf www/

开启权限命令
例如我要开启libao2目录的权限为0777的就下面操作

chmod -R 777 /mnt/libao2/

删除件夹【rm】

rm -rf /mnt/libao2/libao3

将会删除/mnt/libao2/libao3目录以及其下所有文件、文件夹

rm file /mnt/libao2/libao3/123.txt
将会删除 /mnt/libao2/libao3/123.txt文件
